# Decisions

This directory is an append-only architectural decision record (ADR) log.

**When to write here:** Any time you make a decision about technology, structure, or approach that isn't specified in CLAUDE.md, write it here before implementing.

**Format:** See CLAUDE.md → Decisions Log section for the template.

**Numbering:** Sequential. `001-database.md`, `002-state-management.md`, etc.

**Never edit a past decision.** Write a new one that supersedes it and reference the old number.

---

The first required decision is `001-database.md`. Write it before creating any schema files.
